<!-- Schematic -->
### :electric_plug: Schematic
Make the connections according to the table and schematic below.

* ESP32 / st7735 TFT LCD

| ESP32 | TFT Lcd |  
| ----  | ----- |
| 14 | CS   |
| 33 | RST  |
| 27 | DC   |
| 18 | CLK  |
| 23 | DIN  |
| 5V | VCC  |
| 3V3 | LED |
| GND | GND |


* ESP32 / Micro Switch

| Name | GPIO |
| ----  | -----|
| select  | 25    |
| up      | 21    |
| down    | 22    |
